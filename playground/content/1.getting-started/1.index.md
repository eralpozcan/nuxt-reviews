**nuxt-reviews** is a Nuxt module that lets you fetch, normalize, and aggregate customer reviews from multiple platforms through a single, unified API.

## Features

- **Multi-provider support** — Google Places, Trustpilot, SerpAPI, Outscraper, Booking.com :badge[BETA]{type="warning"}
- **Normalized data** — All reviews follow a single `NormalizedReview` type based on Schema.org/Review
- **Auto-imported composable** — `useReviews()` for reactive, type-safe review data
- **Server-side caching** — Configurable TTL to minimize API calls
- **Content moderation** — Built-in Perspective API / OpenAI integration
- **Full TypeScript support** — End-to-end type safety

## How It Works

1. Configure one or more review providers in `nuxt.config.ts`
2. The module registers Nitro server routes (`/api/_reviews`) that fetch and normalize reviews
3. Use the `useReviews()` composable in your Vue components for reactive access
4. Optionally enable content moderation to filter toxic reviews
